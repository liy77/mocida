◆ FontStyle

typedef enum FontStyle FontStyle

Font styles for text rendering.

These styles can be combined using bitwise OR. Normal is the default style. Bold, Italic, and Underscore can be combined with Normal.

◆ UIText

typedef struct UIText UIText

UIText structure representing a text widget.

It contains properties for font size, font family, font style, color, background color, and the text itself.
